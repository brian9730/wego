# 🤖WEGO 프로젝트 
<br>
WEGO는 여행지 추천 웹사이트 입니다.<br>
<br>

## 🛠️개발환경
프론트: Create React App기반으로 React를 사용.<br>
백엔드: Node+Express, 통합개발환경: Visualcode를 사용.<br>

## 💻주요기능
1.오늘의 추천 : 관리자의 추천 장소 리스트를 슬라이드로 제공 <br>
2.축제 및 행사 : 사용 날짜를 기준으로 일주일간의 축제 및 행사를 API를 활용하여 제공 <br>
3.여행 질의 응답 : 여행 관련 질문을 하면 AI가 해당하는 정보를 제공 <br>
4.국내/국외지도 : 오픈스트리트맵 API를 활용한 지역 검색 기능 <br>
5.지역별 관광 정보 : 지역관광정보 API를 활용하여 지도를 보고 선택한 지역의 정보를 카테고리별로 제공<br>
6.지하철 역 주변 정보 : 지하철 노선도에서 원하는 역을 선택, 검색 가능/AI를 활용하여 선택한 역의 맛집, 카페, 관광지를 소개<br>
7.지역 날씨 정보 : 실시간 사용자 위치 또는 원하는 지역의 온도, 습도, 공기질 등을 제공<br>
